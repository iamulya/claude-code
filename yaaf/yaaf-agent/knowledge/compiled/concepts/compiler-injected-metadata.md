## What It Is

Compiler-injected Metadata refers to a set of key-value pairs that the YAAF [Knowledge Compiler](../subsystems/knowledge-compiler.md) automatically adds to the [Frontmatter](./frontmatter.md) of every compiled article [Source 1]. This process ensures that all articles in the knowledge base have a consistent set of core metadata for tracking provenance, type, and compilation status. These fields are not generated by the [LLM](./llm.md) or provided by the user; they are system-level data appended during the final synthesis stage of compilation to enforce structural integrity and provide auditable details about the article's creation [Source 1].

## How It Works in YAAF

During the article synthesis phase, the knowledge compiler's `frontmatter.ts` module is responsible for creating the final frontmatter block. The `buildCompleteFrontmatter` function merges frontmatter from three sources in a specific order of priority:
1. LLM-generated and validated values
2. Compiler-suggested frontmatter from the `ExtractorPlan`
3. Compiler-injected metadata

The compiler-injected metadata has the highest priority and will always overwrite any conflicting keys from the other sources. This guarantees that the final, compiled article has accurate and consistent system-level information [Source 1].

The specific metadata fields injected by the compiler include [Source 1]:

*   `entity_type`: The type of the article (e.g., `concept`, `api`), taken from the `ArticlePlan`.
*   `stub`: A boolean value set to `true` if the article is a stub.
*   `compiled_at`: An ISO timestamp indicating [when](../apis/when.md) the article was compiled.
*   `compiled_from`: A list of source file paths used to generate the article's content.
*   `compiled_from_quality`: An aggregate trust level of the source materials (e.g., C4/A1).
*   `confidence`: A confidence score for the article's content, taken from the `ArticlePlan`.

These fields are always appended after any user- or LLM-generated fields in the final YAML frontmatter block [Source 1].
