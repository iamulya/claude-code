## Overview
`createKBTools` is a factory function used to generate a suite of agent-facing tools for interacting with a compiled Knowledge Base (KB). These tools allow an agent to browse, search, and retrieve document content from a `KBStore` at runtime. 

Unlike Vector-based RAG (Retrieval-Augmented Generation) systems, these tools provide direct document retrieval and keyword searching. The tools generated by this function include:
*   `list_kb_index`: Provides a table of contents for the KB.
*   `fetch_kb_document`: Retrieves the full text of a specific article.
*   `search_kb`: Performs keyword-based searching across all articles in the KB.

## Signature / Constructor

```typescript
export function createKBTools(
  store: KBStore, 
  options: KBToolOptions = {}
): Tool[];

export type KBToolOptions = {
  /** Maximum characters for fetch_kb_document results. Default: 16000 */
  maxDocumentChars?: number
  /** Maximum characters for search_kb excerpts. Default: 800 */
  maxExcerptChars?: number
  /** Maximum search results returned. Default: 5 */
  maxSearchResults?: number
}
```

### Parameters
*   `store`: An instance of `KBStore` representing the loaded Knowledge Base.
*   `options`: An optional configuration object of type `KBToolOptions` to tune the behavior of the generated tools.

## Methods & Properties
The function returns an array of `Tool` objects. Each tool is designed for a specific interaction pattern:

| Tool Name | Description |
| :--- | :--- |
| `list_kb_index` | Returns the `llms.txt`-style table of contents for the Knowledge Base. |
| `fetch_kb_document` | Returns the full content of a specific article. Output is truncated based on `maxDocumentChars`. |
| `search_kb` | Performs keyword search across all articles, returning excerpts and metadata. |

## Examples

### Basic Usage
This example demonstrates how to load a Knowledge Base and provide its tools to a YAAF agent.

```typescript
import { KBStore, createKBTools, Agent } from 'yaaf';

// 1. Initialize and load the store
const store = new KBStore('./my-kb');
await store.load();

// 2. Create the tools
const tools = createKBTools(store, {
  maxDocumentChars: 20000,
  maxSearchResults: 3
});

// 3. Pass tools to the agent
const agent = new Agent({ 
  name: 'KB-Assistant',
  tools 
});
```
