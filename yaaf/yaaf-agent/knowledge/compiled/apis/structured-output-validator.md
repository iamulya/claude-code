## Overview
The `StructuredOutputValidator` is a security and data integrity utility designed to validate and constrain the outputs generated by Large Language Models (LLMs). It ensures that structured data (such as JSON) returned by an agent adheres to a predefined schema, preventing downstream systems from processing malformed, unexpected, or hallucinated data.

Key capabilities include:
*   **JSON Validation**: Verifies fields against specific types and rules.
*   **URL Sanitization**: Validates and filters URLs within the output text or JSON fields.
*   **Constraint Enforcement**: Supports numeric ranges, string length limits, and enum/allowed-value checks.
*   **Violation Handling**: Configurable actions for when validation fails, including logging warnings, stripping invalid fields, or rejecting the entire response.
