## Overview
`CompactionPromptConfig` is a configuration type used to customize the generation of structured compaction prompts. These prompts are designed for production-grade LLM summarization, incorporating an analysis scratchpad and an anti-tool preamble to ensure the model focuses on high-quality conversation synthesis rather than executing functions.

By default, the prompts generated using this configuration produce summaries containing nine required sections. This type allows developers to toggle between full and partial summarization, define custom summary sections, and provide additional instructions to the model.

## Signature / Constructor

```typescript
export type CompactionPromptConfig = {
  /**
   * If true, generates a "partial" prompt that only summarizes recent messages
   * (used when old messages are preserved and only the middle is compacted).
   * If false, generates a "full" prompt that summarizes the entire conversation.
   */
  partial?: boolean;
  /**
   * Custom sections to include in the summary. If omitted, uses all 9 defaults.
   */
  sections?: string[];
  /**
   * Extra instructions appended to the prompt.
   */
  additionalInstructions?: string;
};
```

## Methods & Properties

| Property | Type | Description |
| :--- | :--- | :--- |
| `partial` | `boolean` | (Optional) Determines the scope of the summary. When `true`, the prompt is tailored for "middle-compaction" scenarios where only recent history is summarized. Defaults to `false` (full conversation summary). |
| `sections` | `string[]` | (Optional) An array of strings defining specific headers or categories the model must include in its summary. If not provided, the system uses nine default sections. |
| `additionalInstructions` | `string` | (Optional) Supplemental text or constraints appended to the end of the generated prompt to guide the model's summarization behavior. |

## Examples

### Basic Usage
Generating a full compaction prompt with default settings.

```typescript
import { buildCompactionPrompt, stripAnalysisBlock } from './compactionPrompts.js';

const config: CompactionPromptConfig = {
  partial: false
};

const prompt = buildCompactionPrompt(config);

// Example usage with a model
const summary = await model.complete({ 
  messages: [
    { role: 'user', content: prompt }
  ] 
});

// Remove the <analysis> scratchpad to get the clean summary
const cleanSummary = stripAnalysisBlock(summary.content);
```

### Partial Compaction with Custom Instructions
Configuring a prompt for partial history summarization with specific focus areas.

```typescript
const config: CompactionPromptConfig = {
  partial: true,
  sections: ['User Intent', 'Technical Constraints', 'Resolved Issues'],
  additionalInstructions: 'Focus specifically on any API keys or credentials mentioned.'
};

const prompt = buildCompactionPrompt(config);
```
