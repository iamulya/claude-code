## Overview

The `ModelUsage` type is a data structure that holds the aggregated usage statistics for a single language model within a `CostTracker` instance [Source 1]. It serves as the primary record for tracking all activity related to a specific model, such as "gpt-4o".

Each `ModelUsage` object accumulates the total number of input, output, and cache tokens, the total number of API calls made, and the estimated total cost in USD for that model's usage. These objects are stored within the `models` property of a `CostSnapshot`, allowing for the persistence and restoration of detailed usage data across sessions [Source 1].

## Signature

`ModelUsage` is a TypeScript type alias with the following structure [Source 1]:

```typescript
export type ModelUsage = {
  /** Total input tokens sent to this model. */
  inputTokens: number;

  /** Total output tokens generated by this model. */
  outputTokens: number;

  /** Total tokens retrieved from a cache for this model. */
  cacheReadTokens: number;

  /** Total tokens written to a cache for this model. */
  cacheWriteTokens: number;

  /** Estimated total cost in USD for all calls to this model. */
  costUSD: number;

  /** The total number of times this model was invoked. */
  calls: number;
};
```

## Examples

While `ModelUsage` is not typically instantiated directly by a user, it is a key component of the data returned by a `CostTracker`. For example, [when](./when.md) saving a session's state using `CostTracker.save()`, the resulting `CostSnapshot` object contains a record of `ModelUsage` objects.

```typescript
import { CostTracker, CostSnapshot } from 'yaaf';

// Assume a CostTracker has been used
const tracker = new CostTracker();
tracker.record('gpt-4o', { inputTokens: 1000, outputTokens: 500 });
tracker.record('gpt-4o', { inputTokens: 2000, outputTokens: 1000 });
tracker.record('gpt-4o-mini', { inputTokens: 5000, outputTokens: 200 });

// Get a snapshot of the current usage data
const snapshot: CostSnapshot = tracker.save();

// The `snapshot.models` property contains ModelUsage objects
console.log(snapshot.models);
/*
Output:
{
  'gpt-4o': {
    inputTokens: 3000,
    outputTokens: 1500,
    cacheReadTokens: 0,
    cacheWriteTokens: 0,
    costUSD: 0.0375, // Example cost
    calls: 2
  },
  'gpt-4o-mini': {
    inputTokens: 5000,
    outputTokens: 200,
    cacheReadTokens: 0,
    cacheWriteTokens: 0,
    costUSD: 0.0087, // Example cost
    calls: 1
  }
}
*/

// Accessing a specific model's usage
const gpt4oUsage = snapshot.models['gpt-4o'];
console.log(`GPT-4o was called ${gpt4oUsage.calls} times.`);
// Output: GPT-4o was called 2 times.
```
