## Overview

The `appendQualityRecord` function is a utility for persisting compile-time quality metrics. It takes a `[[[[[[[[CompileQualityRecord]]]]]]]]` object and appends it as a single-line JSON object to a history file named `.kb-quality-history.jsonl` within the specified knowledge base directory [Source 1].

This function is a key part of the framework's quality tracking system. By creating an append-only log, it enables historical analysis of knowledge base quality, such as tracking grounding scores, lint errors, and compile performance over time. The file is created if it does not already exist. The use of the JSONL format ensures that new records can be added without breaking existing readers or requiring schema migrations [Source 1].

## Signature

```typescript
export async function appendQualityRecord(
  kbDir: string,
  record: CompileQualityRecord,
): Promise<void>;
```

**Parameters**

*   `kbDir` [string]: The path to the root directory of the knowledge base. The `.kb-quality-history.jsonl` file will be written in this directory.
*   `record` [CompileQualityRecord]: An object containing the quality metrics from a single compile run. See the `CompileQualityRecord` type definition for its structure.

**Returns**

*   `Promise<void>`: A promise that resolves [when](./when.md) the record has been successfully appended to the file.

## Examples

The following example demonstrates how to create a sample quality record and append it to the history file for a knowledge base located at `./my-kb`.

```typescript
import { appendQualityRecord } from 'yaaf';
import type { CompileQualityRecord } from 'yaaf';
import { join } from 'path';

async function logCompileMetrics() {
  const kbDirectory = './my-kb';

  // This record would typically be generated by the compiler.
  const compileRecord: CompileQualityRecord = {
    compiledAt: new Date().toISOString(),
    durationMs: 12345,
    articlesCompiled: 50,
    articlesSkipped: 150,
    stubsCreated: 5,
    articlesFailed: 1,
    grounding: {
      articlesVerified: 49,
      articlesPassed: 48,
      articlesFailed: 1,
      meanScore: 0.95,
      minScore: 0.78,
    },
    lint: {
      totalIssues: 12,
      errors: 2,
      warnings: 10,
      topCodes: [{ code: 'missing-summary', count: 7 }],
    },
    size: {
      totalArticles: 200,
      totalSources: 85,
    },
  };

  try {
    await appendQualityRecord(kbDirectory, compileRecord);
    console.log('Successfully appended quality record.');
    // After this runs, a new line with the JSON of `compileRecord`
    // will be added to `./my-kb/.kb-quality-history.jsonl`.
  } catch (error) {
    console.error('Failed to write quality record:', error);
  }
}

logCompileMetrics();
```
