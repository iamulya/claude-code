## Overview

`ContentReplacementSnapshot` is a TypeScript type that defines the data structure for a point-in-time snapshot of file edits made by an agent. This snapshot is typically generated by a `ContentReplacementTracker` instance.

The primary purpose of this snapshot is to preserve the history of file modifications across [Context Compaction](../concepts/context-compaction.md) events. [when](./when.md) an agent's message history is shortened or compacted to save space, the record of which files were edited can be lost. By creating a `ContentReplacementSnapshot` before compaction, this state can be saved and later re-injected into the agent's context, ensuring the agent remains aware of the work it has done. [Source 1]

## Signature

`ContentReplacementSnapshot` is an object containing a record of file edits and a timestamp.

```typescript
export type ContentReplacementSnapshot = {
  /**
   * A record where keys are file paths and values are FileEdit objects
   * describing the last known state of edits for that file.
   */
  files: Record<string, FileEdit>;

  /**
   * An ISO 8601 string representing when the snapshot was created.
   */
  savedAt: string;
};
```

### Constituent Types

The `ContentReplacementSnapshot` type relies on the following supporting types:

```typescript
/**
 * Describes a single tracked edit to a file.
 */
export type FileEdit = {
  type: EditType;
  summary: string;
  timestamp: number;
  /** Total number of edits to this file. */
  editCount: number;
};

/**
 * The type of file modification.
 */
export type EditType = "create" | "modify" | "delete" | "rename";
```

## Examples

Below is an example of what a `ContentReplacementSnapshot` object might look like in practice, representing edits to two files.

```typescript
const snapshot: ContentReplacementSnapshot = {
  files: {
    'src/auth.ts': {
      type: 'modify',
      summary: 'Added null check at line 42',
      timestamp: 1678886400000,
      editCount: 3,
    },
    'src/newFeature.ts': {
      type: 'create',
      summary: 'Created new file for feature X',
      timestamp: 1678886340000,
      editCount: 1,
    },
  },
  savedAt: '2023-03-15T12:00:00.000Z',
};
```
