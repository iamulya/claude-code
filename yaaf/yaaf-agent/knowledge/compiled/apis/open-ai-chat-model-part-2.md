## Constructor

The `OpenAIChatModel` is instantiated with a configuration object of type `OpenAIModelConfig`.

```typescript
import { OpenAIChatModel } from 'yaaf';

const model = new OpenAIChatModel({
  apiKey: process.env.OPENAI_API_KEY!,
  model: 'gpt-4o-mini',
});
```
