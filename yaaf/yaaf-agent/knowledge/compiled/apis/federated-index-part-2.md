## Signature

`FederatedIndex` is a TypeScript type alias. Its structure is defined as follows [Source 1]:

```typescript
export type FederatedIndex = {
  /** Total documents across all KBs */
  totalDocuments: number;

  /** Total estimated tokens across all KBs */
  totalTokenEstimate: number;

  /** All entries with namespace info */
  entries: NamespacedIndexEntry[];

  /** Per-namespace stats */
  namespaces: Array<{
    namespace: string;
    label: string;
    documentCount: number;
    tokenEstimate: number;
  }>;
};
```
