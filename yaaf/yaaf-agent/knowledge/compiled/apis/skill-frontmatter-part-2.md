## Signature

`SkillFrontmatter` is a type alias for an object with the following properties [Source 1]:

```typescript
export type SkillFrontmatter = {
  /** Display name for the skill */
  name: string;
  /** Short description shown in skill listings */
  description?: string;
  /** Version string */
  version?: string;
  /** Whether this skill is always injected (default: true) */
  always?: boolean;
  /** List of tags for filtering/search */
  tags?: string[];
};
```
