## Overview
`chunkResponse` is a utility function used to ensure that agent-generated text conforms to the message length constraints of various communication platforms. When an agent produces a response that exceeds a channel's maximum character limit, this function divides the text into an array of smaller strings.

The function is designed to maintain readability by attempting to split text at natural boundaries, such as paragraphs or sentences, rather than cutting through the middle of a word. This is particularly useful when integrating agents with services like Telegram, Discord, or WhatsApp, which have strict per-message character caps.

## Signature / Constructor

```typescript
export function chunkResponse(text: string, channelName: string): string[]
```

### Parameters
- `text`: The raw string output generated by the agent that needs to be processed.
- `channelName`: A string identifier for the target channel (e.g., `'telegram'`, `'discord'`). This identifier is used to look up the specific character limits associated with that platform.

### Returns
Returns an array of strings (`string[]`), where each element represents a chunk of the original text that fits within the target channel's limits.

## Examples

### Basic Usage
This example demonstrates how to manually use `chunkResponse` to prepare a long string for a specific channel.

```typescript
import { chunkResponse } from 'yaaf';

const longResponse = "This is a very long response from the agent..."; // Assume this is > 4000 chars
const chunks = chunkResponse(longResponse, 'telegram');

for (const chunk of chunks) {
  await telegramChannel.send({ text: chunk });
}
```

### Integration with Gateway
`chunkResponse` is often used within the `afterProcess` hook of a `GatewayConfig` to automatically handle long outputs.

```typescript
const gatewayConfig = {
  agent: myAgent,
  channels: [telegramChannel],
  afterProcess: (response, message) => {
    // Automatically chunk the response based on the inbound message's channel
    return chunkResponse(response, message.channel);
  }
};
```
