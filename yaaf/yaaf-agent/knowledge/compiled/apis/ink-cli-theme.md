## Overview

`Ink[[[[[[[[CLI]]]]]]]]Theme` is a TypeScript type that defines the color scheme for the premium, React-based terminal user interface generated by the `createInkCLI` function [Source 1]. It allows developers to customize the appearance of their agent's command-line interface by specifying colors for different UI elements.

Unlike the `CLITheme` used by the zero-dependency `createCLI` function which uses raw ANSI escape codes, `InkCLITheme` uses human-readable string names for colors (e.g., `'cyan'`, `'magenta'`) that are compatible with the Ink framework [Source 1].

## Signature

`InkCLITheme` is an object type where each key represents a semantic UI element and its value is a string representing an Ink-compatible color name [Source 1].

```typescript
export type InkCLITheme = {
  primary?: string;
  secondary?: string;
  accent?: string;
  error?: string;
  dim?: string;
};
```

| Property    | Type     | Description                               |
| ----------- | -------- | ----------------------------------------- |
| `primary`   | `string` | The main color, often used for agent name or user input. |
| `secondary` | `string` | A secondary color, often used for agent responses. |
| `accent`    | `string` | A color for highlighting important information, like successful [Tool Calls](../concepts/tool-calls.md). |
| `error`     | `string` | The color used for displaying error messages. |
| `dim`       | `string` | A subdued color for less important text, like metadata or footers. |

## Examples

The following example demonstrates how to apply a custom theme [when](./when.md) creating an Ink-based CLI for a YAAF agent. The theme is passed in the configuration object to `createInkCLI` [Source 1].

```typescript
import { Agent, toStreamableAgent } from 'yaaf';
import { createInkCLI } from 'yaaf/cli-ink';

const agent = new Agent({
  systemPrompt: 'You are a helpful assistant.',
  // ... tools, etc.
});

createInkCLI(toStreamableAgent(agent), {
  name: 'my-bot',
  greeting: '👋 Hello! How can I help?',
  theme: {
    primary: 'cyan',
    secondary: 'magenta',
    accent: 'green',
    error: 'red',
    dim: 'gray',
  },
});
```
