## Overview

`CamoufoxConfig` is a TypeScript type alias that defines the configuration object for the `CamoufoxPlugin`. This object is passed to the `CamoufoxPlugin` constructor to customize the behavior of the underlying stealth browser instance it manages.

It allows for detailed control over various aspects of the browser, including its startup mode (e.g., headless), networking (e.g., proxy settings), stealth features (e.g., fingerprinting, geolocation), performance optimizations (e.g., blocking images), and how the plugin integrates with the agent's [Tools](../subsystems/tools.md)et (e.g., custom tool name prefixes).

## Signature

`CamoufoxConfig` is a type alias for an object with the following properties:

```typescript
export type CamoufoxConfig = {
  pythonPath?: string;
  host?: string;
  port?: number;
  headless?: boolean;
  virtualDisplay?: boolean;
  proxy?: { server: string; username?: string; password?: string };
  fingerprint?: Record<string, unknown>;
  geolocation?: { latitude: number; longitude: number };
  locale?: string;
  blockImages?: boolean;
  blockWebRTC?: boolean;
  addons?: string[];
  startupTimeoutMs?: number;
  /** Prefix for generated tool names (default: 'web') */
  toolPrefix?: string;
  /** Auto-start browser on plugin init (default: false) */
  autoStart?: boolean;
};
```

### Properties

*   `pythonPath?: string`: The file path to the Python executable used to launch the Camoufox server.
*   `host?: string`: The hostname or IP address for the Camoufox server to listen on.
*   `port?: number`: The port for the Camoufox server to listen on.
*   `headless?: boolean`: If `true`, the browser will run in headless mode without a graphical user interface.
*   `virtualDisplay?: boolean`: If `true`, uses a virtual display (like Xvfb) for headless mode, which can be necessary on some systems.
*   `proxy?: { server: string; username?: string; password?: string }`: Configures the browser to route traffic through a proxy server.
*   `fingerprint?: Record<string, unknown>`: An object specifying browser fingerprint overrides to enhance stealth.
*   `geolocation?: { latitude: number; longitude: number }`: Sets the browser's geographical location.
*   `locale?: string`: Sets the browser's locale (e.g., `en-US`).
*   `blockImages?: boolean`: If `true`, prevents the browser from loading images, which can speed up page loads.
*   `blockWebRTC?: boolean`: If `true`, disables WebRTC to prevent potential IP address leaks.
*   `addons?: string[]`: A list of paths to browser extensions (addons) to be installed on startup.
*   `startupTimeoutMs?: number`: The maximum time in milliseconds to wait for the browser process to start.
*   `toolPrefix?: string`: A prefix for the names of the Tools generated by this plugin. The default is `'web'`, resulting in tools like `web_browse`, `web_click`, etc.
*   `autoStart?: boolean`: If `true`, the browser process is started automatically [when](./when.md) the plugin is initialized. Defaults to `false`.

## Examples

The following examples demonstrate how to create a `CamoufoxConfig` object and pass it to the `CamoufoxPlugin`.

### Basic Headless Configuration

This example configures the plugin to run a headless browser and start it automatically upon registration.

```typescript
import { PluginHost } from 'yaaf';
import { CamoufoxPlugin, type CamoufoxConfig } from 'yaaf/plugins';

const headlessConfig: CamoufoxConfig = {
  headless: true,
  autoStart: true,
};

async function setupAgent() {
  const host = new PluginHost();
  await host.register(new CamoufoxPlugin(headlessConfig));
  
  // The browser is now running in the background
}
```

### Advanced Configuration with Proxy and Custom Tools

This example shows a more complex setup using a proxy, setting a specific geolocation, increasing the startup timeout, and customizing the tool names.

```typescript
import { PluginHost } from 'yaaf';
import { CamoufoxPlugin, type CamoufoxConfig } from 'yaaf/plugins';

const advancedConfig: CamoufoxConfig = {
  headless: true,
  autoStart: true,
  proxy: {
    server: 'http://user:pass@proxy.example.com:8080',
  },
  geolocation: {
    latitude: 34.0522,
    longitude: -118.2437, // Los Angeles
  },
  toolPrefix: 'browser', // Tools will be named browser_browse, browser_click, etc.
  startupTimeoutMs: 60000, // 1 minute
};

async function setupAgentWithProxy() {
  const host = new PluginHost();
  await host.register(new CamoufoxPlugin(advancedConfig));

  const tools = host.getAllTools();
  // tools will include tools named 'browser_browse', 'browser_click', etc.
}
```
