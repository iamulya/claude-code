## Overview
`CamoufoxConfig` is a TypeScript type definition that specifies the configuration parameters for the `CamoufoxPlugin`. It allows developers to customize the behavior of the stealth browser automation integration, including network settings (proxies), browser environment (fingerprints, geolocation, locale), and plugin-specific behavior like tool naming and initialization.

## Signature / Constructor

```typescript
export type CamoufoxConfig = {
  pythonPath?: string
  host?: string
  port?: number
  headless?: boolean
  virtualDisplay?: boolean
  proxy?: { server: string; username?: string; password?: string }
  fingerprint?: Record<string, unknown>
  geolocation?: { latitude: number; longitude: number }
  locale?: string
  blockImages?: boolean
  blockWebRTC?: boolean
  addons?: string[]
  startupTimeoutMs?: number
  /** Prefix for generated tool names (default: 'web') */
  toolPrefix?: string
  /** Auto-start browser on plugin init (default: false) */
  autoStart?: boolean
}
```

## Methods & Properties

| Property | Type | Description |
| :--- | :--- | :--- |
| `pythonPath` | `string` | Optional path to the Python executable required by the underlying Camoufox driver. |
| `host` | `string` | The host address for the browser instance. |
| `port` | `number` | The port number for the browser instance. |
| `headless` | `boolean` | Whether to run the browser in headless mode. |
| `virtualDisplay` | `boolean` | Whether to use a virtual display (often used in Linux environments for headless GUI support). |
| `proxy` | `object` | Configuration for proxy settings, including `server`, `username`, and `password`. |
| `fingerprint` | `Record<string, unknown>` | Custom browser fingerprint data to evade detection. |
| `geolocation` | `object` | Latitude and longitude coordinates to spoof the browser's location. |
| `locale` | `string` | The locale string (e.g., 'en-US') for the browser. |
| `blockImages` | `boolean` | If true, prevents the browser from loading images to save bandwidth and speed up execution. |
| `blockWebRTC` | `boolean` | If true, disables WebRTC to prevent IP leaks. |
| `addons` | `string[]` | A list of paths to browser extensions/addons to load. |
| `startupTimeoutMs` | `number` | Timeout in milliseconds for the browser process to start. |
| `toolPrefix` | `string` | Prefix applied to the names of tools generated by the plugin. Defaults to `'web'`. |
| `autoStart` | `boolean` | Whether to automatically launch the browser process when the plugin is initialized. Defaults to `false`. |

## Examples

### Basic Configuration
```typescript
import { CamoufoxPlugin, type CamoufoxConfig } from 'yaaf';

const config: CamoufoxConfig = {
  headless: true,
  toolPrefix: 'browser',
  autoStart: true
};

const plugin = new CamoufoxPlugin(config);
```
