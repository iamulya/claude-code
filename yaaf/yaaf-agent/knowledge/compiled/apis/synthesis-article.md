## Overview

The `SynthesisArticle` type represents the structure of a complete markdown article generated by a Large Language Model ([LLM](../concepts/llm.md)) during the knowledge base compilation process [Source 1]. It is derived from the `SynthesisArticleSchema`, a Zod schema used to validate the output from the [LLM Synthesis](../concepts/llm-synthesis.md) step.

This type serves as a data contract at the boundary between the YAAF [Knowledge Base Compiler](../subsystems/knowledge-base-compiler.md) and the untrusted output of an LLM. It ensures that any generated article has a valid metadata wrapper, including essential fields like `docId`, `title`, and `entityType`, before it is processed further. The validation focuses on the metadata and the presence of markdown content, not the correctness of the markdown body itself [Source 1].

## Signature

`SynthesisArticle` is a TypeScript type inferred from a Zod schema.

```typescript
export type SynthesisArticle = z.infer<typeof SynthesisArticleSchema>;
```

The underlying `SynthesisArticleSchema` that defines the structure and validation rules is as follows [Source 1]:

```typescript
import { z } from "zod";

export const SynthesisArticleSchema = z.object({
  docId: z.string().min(1),
  title: z.string().min(1),
  entityType: z.string().min(1),
  markdown: z.string().min(10),
  confidence: z.number().min(0).max(1).default(0.5),
  sources: z.array(z.string()).default([]),
});
```

## Properties

An object of type `SynthesisArticle` has the following properties:

| Property     | Type       | Description                                                                                             |
|--------------|------------|---------------------------------------------------------------------------------------------------------|
| `docId`      | `string`   | A unique identifier for the document. Must be at least one character long.                              |
| `title`      | `string`   | The title of the article. Must be at least one character long.                                          |
| `entityType` | `string`   | The type of entity the article describes (e.g., 'api', 'concept'). Must be at least one character long. |
| `markdown`   | `string`   | The full markdown content of the article body. Must be at least 10 characters long.                     |
| `confidence` | `number`   | A score between 0.0 and 1.0 indicating the LLM's confidence in the generated content. Defaults to `0.5`.  |
| `sources`    | `string[]` | An array of source identifiers used to generate the article. Defaults to an empty array `[]`.           |
