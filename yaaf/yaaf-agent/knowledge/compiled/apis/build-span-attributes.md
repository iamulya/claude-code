## Overview
`buildSpanAttributes` is a utility function used within YAAF's tracing module to generate a standardized set of OpenTelemetry `Attributes`. It ensures that every span produced by the framework contains a consistent set of base dimensions (via `getBaseAttributes`) while allowing for span-specific metadata and custom user-provided attributes.

This function is central to maintaining consistent cardinality dimensions across all traces generated by the framework, such as agent runs, LLM requests, and tool executions.

## Signature / Constructor

```typescript
export function buildSpanAttributes(
  spanType: YAAFSpanType,
  custom: Attributes = {}
): Attributes
```

### Parameters
- `spanType`: A value from the `YAAFSpanType` union identifying the specific operation being traced.
- `custom`: An optional object of type `Attributes` (from `@opentelemetry/api`) containing additional key-value pairs to include in the span.

### Associated Types
The `spanType` parameter must be one of the following string literals defined in `YAAFSpanType`:

| Value | Description |
| :--- | :--- |
| `agent.run` | A single call to `Agent.run()` |
| `llm.request` | A single `ChatModel.complete()` call |
| `tool.call` | A tool invocation attempt (pre-permission check) |
| `tool.execution` | The actual execution of a tool function |
| `tool.blocked` | A tool call denied by permission policy or hooks |
| `memory.extract` | Execution of `MemoryStrategy.extract()` |
| `memory.retrieve` | Execution of `MemoryStrategy.retrieve()` |
| `compaction` | Execution of a context compaction strategy |

## Methods & Properties
The function internally aggregates attributes from three sources:
1.  **Base Attributes**: Standardized attributes provided by `getBaseAttributes()`, which may be influenced by environment variables like `YAAF_OTEL_INCLUDE_AGENT_NAME`.
2.  **Span Type**: The `spanType` argument is included to categorize the span within the trace.
3.  **Custom Attributes**: The `custom` argument allows developers to attach arbitrary metadata to the span.

## Examples

### Basic Usage
Creating attributes for a standard agent run operation.

```typescript
import { buildSpanAttributes } from 'yaaf/telemetry/attributes';

const attributes = buildSpanAttributes('agent.run', {
  'agent.id': 'primary-assistant',
  'session.id': 'sess_98765'
});
```

### Tool Execution Tracing
Generating attributes when a tool is executed within the framework.

```typescript
import { buildSpanAttributes } from 'yaaf/telemetry/attributes';

const toolAttributes = buildSpanAttributes('tool.execution', {
  'tool.name': 'calculator',
  'tool.input': JSON.stringify({ expression: '2 + 2' })
});
```
