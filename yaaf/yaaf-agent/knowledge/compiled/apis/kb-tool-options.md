## Overview

`KBToolOptions` is a TypeScript type that defines a configuration object for the knowledge base [Tools](../subsystems/tools.md) generated by the `createKBTools` function. It allows developers to set limits on the size and quantity of data returned by the tools, such as `fetch_kb_document` and `search_kb`. This is useful for managing [Context Window](../concepts/context-window.md) size, controlling [LLM](../concepts/llm.md) token consumption, and ensuring that agent responses remain concise.

These options are passed as the second argument to the `createKBTools` function [Source 1].

## Signature

The `KBToolOptions` type is defined as follows [Source 1]:

```typescript
export type KBToolOptions = {
  /** Maximum characters for fetch_kb_document results. Default: 16000 */
  maxDocumentChars?: number;
  /** Maximum characters for search_kb excerpts. Default: 800 */
  maxExcerptChars?: number;
  /** Maximum search results returned. Default: 5 */
  maxSearchResults?: number;
};
```

## Properties

- **`maxDocumentChars?: number`**
  - Specifies the maximum number of characters that the `fetch_kb_document` tool will return for a single document's content.
  - **Default**: `16000` [Source 1].

- **`maxExcerptChars?: number`**
  - Specifies the maximum number of characters for the text excerpts returned by the `search_kb` tool.
  - **Default**: `800` [Source 1].

- **`maxSearchResults?: number`**
  - Specifies the maximum number of search results that the `search_kb` tool will return from a query.
  - **Default**: `5` [Source 1].

## Examples

The following example demonstrates how to use `KBToolOptions` to configure the tools created from a `KBStore` instance.

```typescript
import { KBStore, createKBTools, KBToolOptions } from 'yaaf';

// Assume './my-kb' is a path to a compiled knowledge base
const store = new KBStore('./my-kb');
await store.load();

// Define custom options to override the defaults
const options: KBToolOptions = {
  maxSearchResults: 10,       // Return up to 10 search results
  maxDocumentChars: 8000,     // Truncate fetched documents to 8000 characters
  maxExcerptChars: 400,       // Make search excerpts shorter
};

// Create the tools with the specified configuration
const kbTools = createKBTools(store, options);

// These tools can now be provided to a YAAF agent
// const agent = new Agent({ tools: kbTools });
```
