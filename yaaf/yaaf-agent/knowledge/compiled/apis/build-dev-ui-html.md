## Overview

The `buildDevUiHtml` function generates a self-contained HTML page for the YAAF Development UI [Source 1]. This UI is a web-based interface for interacting with and debugging an agent.

This function is primarily used internally by the `createServer()` runtime [when](./when.md) its `devUi: true` option is enabled. It is called once at server startup, and the resulting HTML string is cached and served at the server's root path (`GET /`) [Source 1].

The generated HTML has zero external dependencies. All necessary CSS and JavaScript are inlined into the single file, making it portable and easy to serve [Source 1].

Key features of the Dev UI generated by this function include [Source 1]:
*   Syntax highlighting for various languages (JS/TS, Python, JSON, Bash, CSS).
*   A welcome screen with prompt suggestions.
*   A "jump-to-bottom" button for long conversations.
*   A mobile-friendly inspector drawer.
*   The ability to export conversations as Markdown files.
*   Partial Markdown previews during response [Streaming](../concepts/streaming.md).

## Signature / Constructor

The function takes a single options object and returns a string containing the full HTML document.

```typescript
export function buildDevUiHtml(opts: DevUiOptions): string;
```

### `DevUiOptions` Type

The `opts` parameter must conform to the `DevUiOptions` type [Source 1]:

```typescript
export type DevUiOptions = {
  /** Agent display name */
  name: string;
  /** Agent version */
  version: string;
  /** Whether the agent supports runStream() */
  streaming: boolean;
  /** Model identifier (shown in inspector). Null = not known. */
  model: string | null;
  /** Whether server-side multi-turn history formatting is active */
  multiTurn: boolean;
  /** System prompt to show read-only in Settings. Null = not exposed. */
  systemPrompt: string | null;
};
```

## Examples

The following example demonstrates how to call `buildDevUiHtml` with a sample configuration to generate the UI's HTML.

```typescript
import { buildDevUiHtml, DevUiOptions } from 'yaaf';
import { writeFileSync } from 'node:fs';

// 1. Define the options for the Dev UI
const devUiOptions: DevUiOptions = {
  name: 'My Research Agent',
  version: '1.2.3',
  streaming: true,
  model: 'claude-3-5-sonnet-20240620',
  multiTurn: true,
  systemPrompt: 'You are a helpful research assistant.',
};

// 2. Generate the HTML string
const htmlContent = buildDevUiHtml(devUiOptions);

// 3. (Optional) Save the HTML to a file to inspect it
writeFileSync('dev-ui.html', htmlContent);

console.log('Dev UI HTML generated and saved to dev-ui.html');
```
