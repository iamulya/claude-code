## Overview
`buildCompleteFrontmatter` is a utility function used during the knowledge synthesis phase of the YAAF compiler. It is responsible for aggregating frontmatter fields from multiple sources into a single, cohesive object that is eventually serialized into the final markdown article.

The function ensures structural consistency by applying a specific priority order during the merge process. It specifically protects compiler-injected metadata fields, ensuring they are not overwritten by LLM-generated content or initial suggestions.

## Signature / Constructor

```typescript
export function buildCompleteFrontmatter(
  validatedValues: Record<string, unknown>,
  suggestedFrontmatter: Record<string, unknown>,
  compilerMeta: {
    entity_type: string;
    stub: boolean;
    compiled_at: string;
    compiled_from: string[];
    confidence: number;
  }
): Record<string, unknown>
```

### Parameters
*   **validatedValues**: An object containing frontmatter fields generated by an LLM that have already passed through `validateFrontmatter`.
*   **suggestedFrontmatter**: Initial frontmatter values derived from the `ExtractorPlan`.
*   **compilerMeta**: A set of mandatory metadata fields injected by the compiler to track the article's provenance and state.

### Priority Order
The function merges fields in the following order of increasing priority:
1.  LLM-generated and validated values.
2.  Compiler-suggested frontmatter from the `ExtractorPlan`.
3.  Compiler-injected metadata (always takes precedence for its specific keys).

## Examples

### Basic Usage
This example demonstrates merging validated LLM output with compiler metadata.

```typescript
import { buildCompleteFrontmatter } from './frontmatter';

const validated = {
  title: "Agent Framework",
  summary: "A high-level overview of agent architectures."
};

const suggested = {
  category: "concept"
};

const meta = {
  entity_type: "concept",
  stub: false,
  compiled_at: "2023-10-27T12:00:00Z",
  compiled_from: ["docs/intro.md", "src/core.ts"],
  confidence: 0.98
};

const finalFrontmatter = buildCompleteFrontmatter(validated, suggested, meta);

/*
Result:
{
  title: "Agent Framework",
  summary: "A high-level overview of agent architectures.",
  category: "concept",
  entity_type: "concept",
  stub: false,
  compiled_at: "2023-10-27T12:00:00Z",
  compiled_from: ["docs/intro.md", "src/core.ts"],
  confidence: 0.98
}
*/
```
