## Signature

The `PerUserRateLimiterConfig` object is passed to the [PerUserRateLimiter](./per-user-rate-limiter.md) constructor or the `perUserRateLimiter` factory function. While the full type definition is not present in the source, its properties can be inferred from usage examples [Source 1].

```typescript
export type PerUserRateLimiterConfig = {
  /**
   * The maximum number of turns a user can consume within the time window.
   */
  maxTurnsPerUser: number;

  /**
   * The duration of the rolling window in milliseconds.
   * For example, 3_600_000 for one hour.
   */
  windowMs: number;

  /**
   * An optional backend for distributed rate limiting across multiple replicas.
   * If not provided, an in-memory backend is used.
   * @see [[DistributedRateLimitBackend]]
   */
  backend?: DistributedRateLimitBackend;

  // Note: While the backend interface tracks 'cost' and 'tokens' in addition to 'turns',
  // explicit configuration properties for these are not shown in the provided source material.
};
```
