## Overview

The `parseArticleOutput` function is a utility for parsing the raw string output generated by a Large Language Model ([LLM](../concepts/llm.md)) into a structured article object. It is designed to handle markdown content that is expected to begin with a YAML [Frontmatter](../concepts/frontmatter.md) block, delimited by `---` [Source 1].

The primary use case for this function is in the YAAF [Knowledge Base Compiler](../subsystems/knowledge-base-compiler.md), where it serves as the initial step in processing an LLM-synthesized article. It separates the machine-readable metadata (frontmatter) from the human-readable content (body) [Source 1].

A key feature of this function is its robustness. Although the LLM is prompted to include a frontmatter block, it may occasionally fail to do so. In such cases, `parseArticleOutput` does not throw an error. Instead, it gracefully handles the omission by returning an empty frontmatter object and treating the entire input string as the article body [Source 1].

## Signature

The function takes a single string argument and returns a `ParsedArticle` object.

```typescript
import type { ParsedArticle } from "./types.js";

export function parseArticleOutput(raw: string): ParsedArticle;
```

### Parameters

-   `raw` (string): The raw markdown string output from the LLM.

### Return Value

-   (`ParsedArticle`): An object containing the parsed frontmatter and the body of the article. The `ParsedArticle` type is defined as:
    ```typescript
    type ParsedArticle = {
      frontmatter: Record<string, unknown>;
      body: string;
    };
    ```

## Examples

### Standard Usage with Frontmatter

This example shows how the function parses a typical LLM output containing both frontmatter and a body.

```typescript
import { parseArticleOutput } from 'yaaf';

const llmOutput = `---
title: "Agent"
entity_type: "concept"
summary: "An autonomous entity that perceives its environment and acts upon it."
---

An **agent** is a fundamental concept in artificial intelligence.
It refers to anything that can be viewed as perceiving its
environment through sensors and acting upon that environment
through actuators.`;

const parsed = parseArticleOutput(llmOutput);

console.log(parsed.frontmatter);
// {
//   title: "Agent",
//   entity_type: "concept",
//   summary: "An autonomous entity that perceives its environment and acts upon it."
// }

console.log(parsed.body);
// "\nAn **agent** is a fundamental concept in artificial intelligence.\nIt refers to anything that can be viewed as perceiving its\nenvironment through sensors and acting upon that environment\nthrough actuators."
```

### Handling Missing Frontmatter

This example demonstrates the function's behavior [when](./when.md) the input string lacks a frontmatter block.

```typescript
import { parseArticleOutput } from 'yaaf';

const llmOutputWithoutFrontmatter = `This is an article body without any frontmatter.
The LLM forgot to include the --- delimiters.`;

const parsed = parseArticleOutput(llmOutputWithoutFrontmatter);

console.log(parsed.frontmatter);
// {}

console.log(parsed.body);
// "This is an article body without any frontmatter.\nThe LLM forgot to include the --- delimiters."
```
