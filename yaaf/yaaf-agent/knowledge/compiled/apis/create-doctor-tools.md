## Overview
`createDoctorTools` is a factory function that generates a suite of code intelligence tools designed for the YAAF Doctor agent. These tools enable an agent to perform diagnostic and development tasks on a YAAF-based project, including reading files, searching the codebase, compiling code, running tests, and inspecting project structure.

The tools generated by this function are sandboxed to the specified project root, ensuring that the agent's operations remain within the boundaries of the developer's project. While these tools are part of the YAAF framework, they are designed to work on any project that utilizes YAAF, not just the framework's own repository.

## Signature / Constructor
```typescript
export function createDoctorTools(projectRoot: string): Tool[];
```

### Parameters
- `projectRoot`: A string representing the absolute path to the root directory of the project the tools will inspect and manage.

### Returns
- `Tool[]`: An array of `Tool` objects compatible with YAAF agents.

## Examples

### Initializing Doctor Tools
This example shows how to initialize the doctor tools for the current working directory and provide them to an agent.

```typescript
import { createDoctorTools } from 'yaaf/doctor';
import path from 'path';

// Define the project root
const projectRoot = path.resolve(process.cwd());

// Generate the suite of tools
const doctorTools = createDoctorTools(projectRoot);

// The resulting tools can be used when initializing an Agent
// const agent = new Agent({ tools: doctorTools, ... });
```
