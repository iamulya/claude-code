## Overview
`MailboxMessage` is the core data structure used by the YAAF Mailbox system to facilitate communication between agents. It defines the schema for messages stored in file-based inboxes, supporting both in-process communication and multi-process swarms (such as agents running in separate tmux panes). 

Because the Mailbox system is designed for persistence and cross-boundary IPC, this type ensures that messages contain the necessary metadata to survive agent restarts and provide context to recipients without requiring shared memory.

## Signature / Constructor
```typescript
export type MailboxMessage = {
  /** Sender agent name */
  from: string
  /** Message content (text or JSON-serialized structured message) */
  text: string
  /** ISO 8601 timestamp */
  timestamp: string
  /** Whether the recipient has read this message */
  read: boolean
  /** Optional sender color for UI display */
  color?: string
  /** Optional short summary for preview */
  summary?: string
}
```

## Methods & Properties
| Property | Type | Description |
| :--- | :--- | :--- |
| `from` | `string` | The unique identifier or name of the agent that sent the message. |
| `text` | `string` | The primary payload of the message. This can be a raw string or a serialized JSON object for structured data exchange. |
| `timestamp` | `string` | An ISO 8601 formatted string indicating when the message was created. |
| `read` | `boolean` | A status flag used by the Mailbox system to track whether the recipient has acknowledged or processed the message. |
| `color` | `string` | (Optional) A visual hint, typically a hex code or terminal color name, used for rendering the message in logs or UIs. |
| `summary` | `string` | (Optional) A condensed version of the message content, useful for notifications or preview lists. |

## Examples
### Creating a Manual Message
While messages are typically generated by the `Mailbox.send` method, they follow this structure:

```typescript
const message: MailboxMessage = {
  from: 'researcher-agent',
  text: 'Analysis complete. Found 5 critical vulnerabilities.',
  timestamp: new Date().toISOString(),
  read: false,
  color: 'red',
  summary: 'Vulnerability report'
};
```

### Usage in Mailbox Operations
The `Mailbox` class returns arrays of these objects when reading an inbox.

```typescript
// Example of how MailboxMessage is consumed from the Mailbox
const unreadMessages: MailboxMessage[] = await mailbox.readUnread('coordinator', 'security-team');

unreadMessages.forEach((msg) => {
  console.log(`[${msg.timestamp}] ${msg.from}: ${msg.text}`);
});
```
