## Overview
`GeminiChatModel` is a specialized implementation of the `LLMAdapter` interface designed to interface with Google's Gemini family of models. It extends the `BaseLLMAdapter` class and provides a unified interface for both Google AI Studio and Google Cloud Vertex AI.

The class handles the translation of YAAF message formats into the schema required by the Google Generative AI SDK. It supports both standard unary completions and real-time streaming of responses.

### Prerequisites
This model requires the `@google/genai` package (version 0.7.0 or higher) to be installed as a peer dependency:
```bash
npm install @google/genai
```

## Signature / Constructor

### GeminiModelConfig
The configuration object supports two primary authentication modes: Google AI Studio (API Key) and Vertex AI (Project/Location).

## Methods & Properties

### complete()
Executes a standard non-streaming completion request.
- **Returns**: `Promise<LLMResponse>`

### stream()
Executes a streaming completion request, yielding chunks as they are generated by the model.
- **Returns**: `AsyncIterableIterator<LLMResponseChunk>`

### Message Translation
The class includes internal logic to translate standard message arrays into the format expected by the `@google/genai` SDK, handling roles and content structures automatically.

## Examples

### Google AI Studio Usage
```typescript
import { GeminiChatModel } from 'yaaf';

const model = new GeminiChatModel({ 
  apiKey: process.env.GEMINI_API_KEY!,
  model: 'gemini-1.5-pro'
});

const response = await model.complete({
  messages: [{ role: 'user', content: 'Explain quantum entanglement.' }]
});
```

### Vertex AI Usage
```typescript
import { GeminiChatModel } from 'yaaf';

const model = new GeminiChatModel({
  vertexAI: true,
  project: 'my-gcp-project',
  location: 'us-central1',
  model: 'gemini-1.5-flash',
});
```

### Streaming Responses
```typescript
const model = new GeminiChatModel({ apiKey: '...' });

for await (const delta of model.stream({ messages })) {
  process.stdout.write(delta.content ?? '');
}
```

### Plugin Registration
`GeminiChatModel` can be registered with a YAAF host to provide LLM capabilities to the framework.
```typescript
await host.register(new GeminiChatModel({ apiKey: '...' }));
const model = host.getLLMAdapter()!;
```
