## Overview
`LintReport` is a TypeScript type that represents the final result of a linting operation within the YAAF knowledge base compiler. It serves as a container for all issues—such as errors, warnings, or information messages—detected during the analysis of knowledge base articles and their relationships.

This type is used to programmatically evaluate the health of a knowledge base, allowing developers to determine if the content meets the required standards for production deployment or if manual intervention is required to fix structural or content-related issues.

## Signature
`LintReport` is exported as a type from the linter's core type definitions.

```typescript
export type LintReport = {
  // Aggregated issues and results from the linting process
};
```
