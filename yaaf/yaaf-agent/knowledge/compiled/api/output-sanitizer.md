## Overview
`OutputSanitizer` is a security utility designed to mitigate risks associated with OWASP LLM02 (Insecure Output Handling). It processes the text generated by a Large Language Model (LLM) to remove potentially malicious content, such as Cross-Site Scripting (XSS) payloads or unauthorized HTML tags, before the output is rendered in a user interface or processed by downstream systems.

This class is typically used as an `afterLLM` hook within a YAAF agent's lifecycle. It can be used standalone or as part of the broader `securityHooks` composition.

## Signature / Constructor

```typescript
class OutputSanitizer {
  constructor(config?: OutputSanitizerConfig);
}
```

### OutputSanitizerConfig
The configuration object controls the strictness and behavior of the sanitization process.

| Property | Type | Description |
| :--- | :--- | :--- |
| `stripHtml` | `boolean` | If true, removes HTML tags from the LLM response. |

## Methods & Properties

### hook()
Returns a hook function compatible with the `afterLLM` agent hook.

**Signature:**
`hook(): (response: ChatResult, iteration: number) => LLMHookResult | undefined`

**Returns:**
An `LLMHookResult` containing the sanitized text if modifications were made, or `undefined` if no changes were required. If the sanitizer determines the output should be modified, the result will typically have an `action` of `'override'`.

## Examples

### Manual Hook Integration
This example demonstrates how to manually integrate the `OutputSanitizer` into an agent's hook lifecycle.

```typescript
import { Agent, OutputSanitizer } from 'yaaf';

const sanitizer = new OutputSanitizer({ stripHtml: true });

const agent = new Agent({
  systemPrompt: 'You are a helpful assistant.',
  hooks: {
    afterLLM: async (response, iteration) => {
      // Apply sanitization to the LLM response
      const sanitized = sanitizer.hook()(response, iteration);
      
      // If the sanitizer returned an override, use it
      if (sanitized?.action === 'override') {
        return sanitized;
      }
      
      return response;
    },
  },
});
```

### Usage via securityHooks
The `OutputSanitizer` is automatically configured when using the `securityHooks` helper.

```typescript
import { Agent, securityHooks } from 'yaaf';

const agent = new Agent({
  systemPrompt: 'You are a helpful assistant.',
  hooks: securityHooks({
    outputSanitizer: { stripHtml: true }
  }),
});
```
